Welcome Admin,

TEAMPULSE administration is pretty simple. There are two levels of access: TABLE, and ADMIN access.

Users with TABLE privilege are limited to viewing: All Records, Project Tables, and Comments Pages.

Users with ADMIN privilege can view Users Tables, Projects Admin, and Users Admin Pages. These pages can be used to remove individual scores; add or remove projects; and modify user privileges respectively.
